About The Project

SecureRide is a carpooling mobile application designed specifically to address the transportation challenges faced by UTHM students. It was built to solve the issues of high e-hailing costs, limited campus parking, and the safety risks associated with uncertified ride-sharing Telegram groups. It functions as a centralized platform that efficiently matches students who have vehicles with those needing a ride.

Key Features

  • Verified Community: Mandatory student identity verification using the official UTHM email domain to ensure only legitimate students access the platform.
  • Live GPS Tracking: Utilizes real-time GPS to allow passengers to view the driver's live location and share their trip progress with authorized trusted contacts for enhanced safety.
  • Ride-Matching: Enables drivers to post ride offers (route, time, seats) and allows passengers to securely book available rides.
  • Web Admin Portal: Provides administrators with a comprehensive dashboard for user account oversight, verifying users, and monitoring trips.
