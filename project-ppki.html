About The Project

Despite the effectiveness of the Program Pendidikan Khas Integrasi (PPKI) in teaching vocational skills, a significant "transition gap" prevents students with learning disabilities from participating in the real-world digital economy. Mainstream e-commerce platforms are often too complex and fail to meet accessibility standards. Web4Good was developed to bridge this gap, providing an accessible mobile commerce infrastructure that allows students to commercialize their goods while drastically reducing the administrative burden on special education teachers.

Key Features

  • WCAG 2.1 Accessibility: Designed with inclusivity in mind, featuring high-contrast modes, large text options, and simplified navigation for users with cognitive or visual impairments.
  • Role-Based Architecture: Custom dashboards tailored for Buyers (browsing, cart, order history) and Teachers (student management, inventory oversight, financial analytics).
  • Real-Time Cloud Synchronization: Powered by Firebase Firestore to ensure instant inventory deduction, offline caching, and real-time chat between buyers and teachers.
  • Integrated Payment Gateway: Supports both "Pay Cash at Counter" and cashless FPX transactions via a seamless ToyyibPay REST API integration.
  • Interactive Location Tracking: Integrates the OpenStreetMap Tile API to help customers easily locate the physical PPKI school stall.
